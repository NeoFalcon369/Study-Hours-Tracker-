# Study Hours Tracker

A simple web application for students to track study sessions, log hours, and review progress. The app stores data locally in the browser using Local Storage.

## Files

- `index.html` - main page with the form, summary cards, and session table.
- `styles.css` - styling for the layout, form, and table.
- `app.js` - client-side logic for saving, deleting, and displaying study sessions.

## How to use

1. Open `index.html` in a web browser.
2. Enter a subject, date, hours, and optional notes.
3. Click **Add Session**.
4. Your study sessions are saved locally and displayed in the session history.

## Notes

- This app works offline in the browser.
- Sessions are persisted in Local Storage and are available after page refresh.
